# PimpBunny Plugin for Hedon Haven

This is a third-party provider plugin for [Hedon Haven](https://github.com/Hedon-Haven/Hedon-Haven) that integrates PimpBunny.com as a content source.

## Features

- Browse homepage content
- Search functionality
- Video metadata extraction
- Video suggestions
- Author pages
- Comments

## Installation

1. Download the latest `pimpbunny.zip` from [Releases](https://github.com/angel4833/pimpbunny-plugin/releases)
2. Follow the Hedon Haven plugin installation instructions

## Building

```bash
cd src
./compile.sh
```

This will generate `plugin.zip` in the root directory.
